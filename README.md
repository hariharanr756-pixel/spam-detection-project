# 📩 Spam Detection Project

This project uses **Machine Learning (NLP)** to classify messages as **Spam or Not Spam**.

---

## 🚀 Features
- Detects spam messages using text classification
- Uses Natural Language Processing (NLP)
- Simple and efficient model

---

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn

---

## 🤖 Algorithm
- CountVectorizer (Text Processing)
- Multinomial Naive Bayes (Classification)

---

## ▶️ How to Run

1. Install dependencies:
   pip install pandas scikit-learn

2. Run the project:
   python spam_project.py

---

## 📊 Dataset
- Contains labeled messages (spam / ham)
